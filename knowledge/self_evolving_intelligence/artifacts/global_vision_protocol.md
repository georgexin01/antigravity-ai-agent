## 🎯 1. THE MANDATE: LOCAL-FIRST VISION
To save Cloud Tokens and achieve maximum UI verification speed, the AI MUST prioritize **User-Captured Local Screenshots** over its own automated browser captures.

### The "Pulse-Sync" Handshake:
1. **PULSE**: When the AI needs to "see" a state or UI detail, it triggers a user alert: *"READY TO SNIP: [Target Detail]"*.
2. **CAPTURE**: The User takes a screenshot (using **Snipaste** or similar) and saves it to a designated "Hot File" in the current project root.
3. **OBSERVE**: The AI uses `view_file` to read the local PNG/JPG instantly (Zero cloud overhead).
4. **EXECUTE**: The AI analyzes the local visual data and continues the task.
