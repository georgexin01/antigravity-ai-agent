The BLAST Framework ensures the AI Brain operates as a deterministic, multi-agent system rather than a probabilistic prompt-responder.

- **B - BLUEPRINT**: Every project MUST start with a `blueprint.md` defining the "North Star" (Goal), SOPs, and System Rules.
- **L - LINKS**: Use **MCP (Model Context Protocol)** to connect the Brain to external ecosystems (Notion, Calendar, GitHub, Stripe, etc.).
- **A - ARCHITECT**: Three-Layer Decision Logic:
    1. **SOPs (Gold Standard)**: Deterministic business rules in Markdown.
    2. **Reasoning**: Rooting data between SOPs and Tools.
    3. **Atomic Tools**: Discrete Python scripts for testable execution.
- **S - STYLIZE**: Focused UI/UX refinement. Deliver "Stylized Payloads" (React, Notion, Slack Blocks) optimized for zero-friction user consumption.
- **T - TRIGGER**: Autonomous deployment and scheduling via **Modal** or serverless infrastructure.
