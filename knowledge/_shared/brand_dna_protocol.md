# Brand DNA Protocol (V27.3) — Agentic Alignment

The **Brand DNA Audit** is a mandatory P0 step for all new high-fidelity missions. It ensures perfect brand-matching and zero-hallucination design logic.

## 🧬 1. The Brand Persona Card (BPC)

Every new project MUST start with a BPC. This card is generated by the AI and signed-off by the USER.

### BPC Template:
```markdown
# BRAND_DNA_CARD: [Project Name]

## 🏗️ Core Identity
*   **Mission**: [One-sentence mission statement]
*   **Vibe/Intent**: [Cinematic, Industrial, Minimal, Organic, etc.]
*   **Cultural Context**: [e.g., Japanese Tech, JB Engineering, Silicon Valley Minimal]

## 🎨 Design DNA
*   **Primary Base**: [Off-Black/Zinc Code]
*   **Authority Color**: [The "Glow" or Red-equivalent]
*   **Typography A (High Authority)**: [Header Font]
*   **Typography B (System/Mono)**: [Telemetry/Label Font]

## 🎬 Cinematic Intent
*   **Hero Strategy**: [Scroll-Sync Sequence, 3D Point Cloud, or Kinetic Text]
*   **Motion Gravity**: [Stiffness/Damping Settings]
```

## 🔍 2. Extraction Protocol

1.  **URL Input**: If the user provides a reference URL (e.g., `zetacapital.my`), the AI must use `browser_subagent` to read the page.
2.  **Color Sampling**: Identify the dominant colors and convert them to the **Anti-Slop** palette (Zero #000000).
3.  **Typography Analysis**: Identify the existing font family and suggest a "High-Authority" alternative for the V27.3 build.

## 🏁 3. Sign-off Loop

The AI will post the BPC in the chat or as an artifact.
*   **Action**: Wait for user "APPROVE" or "START" before creating any HTML/CSS files.
